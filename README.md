# lol-drx-season-analysis
# DRX 우승 시즌과 비우승 시즌 성과 비교 분석
우승 시즌과 비우승 시즌을 비교하여 성과 차이를 설명하는 요인을 데이터 기반으로 분석한다.

## 1. 프로젝트 개요
본 프로젝트는 리그 오브 레전드 프로팀 DRX를 대상으로,
우승 시즌과 비우승 시즌 간 성과 차이가 발생한 원인을
조합 구조, 로스터 안정성을 중심으로 분석하는 것을 목표로 한다.

게임 내 모든 전술적 판단을 해석하기보다는,
공개 데이터로 관측 가능한 지표를 활용하여
성과 차이를 설명할 수 있는 최소한의 구조를 설계하는 데 초점을 맞춘다.

## 2. 분석 질문
- 우승 시즌과 비우승 시즌 사이에 조합 구조적 차이가 존재하는가?
- 로스터 및 코칭 스태프의 안정성은 시즌 성과와 어떤 관계를 가지는가?

## 3. 데이터 수집
- 대상 게임: League of Legends
- 대상 팀: DRX
- 데이터 범위: 우승 시즌 / 비우승 시즌
- 데이터 출처: 공개 경기 기록 웹사이트
- 수집 방법: Python 기반 웹 크롤링

## 4. 전처리 및 변수 설계
- 챔피언은 개별 비교가 아닌 역할군 단위로 분류
- 상체(Top-Jungle-Mid), 하체(ADC-Support) 구조 중심 조합 정의
- 로스터 안정성 지표 생성 (주전 변경 횟수, 라인업 유지율)

## 5. 분석 방법
- 시즌별 승률 및 지표 비교
- 조합 구조별 승률 분석
- 로지스틱 회귀를 활용한 승패 영향 요인 분석

## 6. 결과 및 인사이트
- 우승 시즌과 비우승 시즌 간 조합 구조 및 운영 지표의 차이를 확인
- 특정 구조적 요소가 승률과 유의미한 관계를 가짐을 확인

## 7. 한계 및 향후 개선
- 패치 버전 차이 미반영
- 선수 개인 기량 요인 미포함
- 인게임 내용 미반영
- 향후 리플레이 데이터 또는 팀 간 비교 확장 가능
